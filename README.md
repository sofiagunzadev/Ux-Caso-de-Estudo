# Ux-Caso-de-Estudo


---

UX - CASO DE ESTUDO 

Landing Page – Workshop de Finanças para Empreendedoras (FINAK)

Projeto: Finanças que Impulsionam Negócios Femininos

Tipo: Landing Page de Evento
Público: Mulheres empreendedoras e futuras empreendedoras
Objetivo: Divulgação e inscrições para um workshop de educação financeira.


---
##  Contexto do Projeto

Muitas mulheres empreendedoras possuem talento para vender e criar negócios, mas enfrentam dificuldades em:

🔹Organizar as finanças;

🔹Definir preços corretamente;

🔹Controlar o fluxo de caixa;

🔹Separar finanças pessoais das empresariais;

🔹Tomar decisões financeiras com segurança.


Foi identificada a oportunidade de criar uma experiência digital que apresentasse o workshop de forma clara, acolhedora e persuasiva, aumentando a taxa de inscrições.


---

## Problema

🔹As páginas de eventos financeiros geralmente:

🔹Utilizam linguagem excessivamente técnica;

🔹Geram intimidação em quem não domina finanças;

🔹Não deixam claro os benefícios do evento;

🔹Possuem excesso de informações;

🔹Não transmitem confiança suficiente para a compra.


---

## Objetivos de Negócio

Primário

Aumentar o número de inscrições no workshop.

Secundários

Fortalecer a autoridade da mentora;

Construir comunidade feminina;

Gerar leads para futuros eventos;

Criar confiança na marca.

---

##  Objetivos de UX

A Landing Page deve:

✓ Comunicar valor rapidamente;

✓ Demonstrar credibilidade;

✓ Reduzir inseguranças;

✓ Facilitar a tomada de decisão;

✓ Tornar a inscrição simples e intuitiva.

16. Aprendizados do Projeto

Linguagem simples gera mais conexão.

Provas sociais aumentam confiança.

Benefícios precisam aparecer antes das funcionalidades.

Uma Landing Page de evento deve transmitir emoção e clareza simultaneamente.

O design não é apenas estética; ele reduz inseguranças e facilita decisões.
